Drag'n Share

Share files in a local network in two drags (completely insecure).

Drag'n Share is a small cross-platform application that lets you share files really easy in a local network. You drag your file on one machine and it appears in a popup on all other machines in the network where you can drag it from.

Dependencies

Java 1.6+
